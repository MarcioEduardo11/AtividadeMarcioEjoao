# TRABALHO SOBRE CRIAÇÃO DE PROJETO EM CONJUNTO

## Trabalho sobre Criação de uma WIKIPEDIA do calouro EM DUPLAS

O projeto consiste em criar, em duplas, uma “Wikipedia do Calouro”, reunindo informações úteis para novos estudantes. Nele, serão apresentados os cursos oferecidos, explicando de forma simples o que cada um aborda. O objetivo é organizar informações importantes e ajudar os calouros a conhecer melhor a instituição.
Onde vamos abordar:
* Dicas
* Discilpinas
